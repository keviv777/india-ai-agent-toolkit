# 🤝 Contributing to India AI Agent Toolkit

We welcome contributions from the community! Whether you have an agent for Indian HR policies, local SaaS customer support, or regional language translation, follow these steps to add it.

## How to Add a New Agent
1. **Fork** this repository.
2. **Create a markdown file** in the `agents/` folder. Use the name format `topic-agent-name.md`.
3. Ensure your agent file includes:
   - A clear Title and Description.
   - Personality & Tone instructions.
   - Process or Capabilities.
   - An exact **System Prompt** example.
4. **Update the `README.md`** to link your new agent.
5. Submit a **Pull Request** with a brief description of what your agent does.

Let's build the best open repo for Indian AI use cases!
